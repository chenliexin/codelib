1.通用
1.1.命名
    项目、目录、文件使用小写命名，下划线分隔，只能使用小写字母、数字、下划线
    示例：project、my_project、jbk_art_2015
1.2.编码
    优先使用utf-8编码
1.3.缩进
    使用tab缩进，设置tab为4个空格

2.HTML
2.1.基本
    嵌套缩进
    示例（嵌套块/行内嵌套）：
        <div class="block">
            <div class="block"><span></span><span></span></div>
            <div class="block">
                <div>text hello world</div>
            </div>
        </div>
    同级并行
    示例：
        <div class="block">
            <h1>title</h1>
            <div class="block">text hello world</div>
        </div>
    示例（反模式）：
        <div class="block">
            <h1>title</h1>
            <img src="logo.png">
        </div>
    闭合：普通标签必须闭合，自动闭合标签结尾处不添加斜杠
    属性：使用双引号，属性名使用小写，自定义属性名只能使用小写字母、数字、短横线
2.2.使用HTML5 doctype
    lang属性是必须
2.3.兼容模式
    示例：
        <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
2.4.引入CSS/JS不需要type属性，因为type已经默认
2.5.属性顺序
    class
    id //如非交互需要，不使用id
    name
    data-*
    src, for, type, href, value, max-length, max, min, pattern
    placeholder, title, alt
    aria-*, role
    required, readonly, disabled, checked, selected //boolean属性指不需要声明取值的属性
2.6.避免使用JS生成标签

3.CSS
3.1.选择器
    选择器主体使用小写命名，短横线分隔，只能使用小写字母、数字、短横线
    选择器嵌套尽可能少
3.2.引号
    优先使用单引号
3.3.属性声明顺序（布局、尺寸、边距、文本、颜色背景、其他）
    display, float, position
    width/height, margin/padding, border
    font, line-height, text-align
    color, background, opacity
3.4.属性简写，并使用缺省值、默认值
    示例：
       font-size: 14px; line-height: 1.5; font-family: simsun; -> font: 14px/1.5 simsun;
       padding-bottom: 10px; -> padding: 0 0 10px;
3.5.媒体查询（media query）
    媒体查询的规则放在对应规则的后面，不要放在其他位置或文件最底部，更不要独立到其他文件
